# IPL Fantasy Pro - UI/UX Design

## Design Philosophy
- **Dark Theme First** - Easy on eyes during long drafts
- **Data-Dense** - Show maximum information without clutter
- **Real-Time Focus** - Everything updates live
- **Cricket Aesthetic** - Green pitches, amber trophies, blue jerseys

## Component Library

### Buttons
```
Primary: bg-blue-600 hover:bg-blue-700 text-white
Secondary: bg-slate-700 hover:bg-slate-600 text-white
Success: bg-green-600 hover:bg-green-700 text-white
Danger: bg-red-600 hover:bg-red-700 text-white
Outline: border border-slate-500 hover:bg-slate-800 text-white
```

### Cards
```
Background: bg-slate-800
Border: border border-slate-700
Border Radius: rounded-xl (12px)
Shadow: shadow-lg
Padding: p-4 or p-6
```

### Player Cards
```
Small: 80x100px - Avatar + Name + Price
Medium: 120x160px - Add stats preview
Large: Full width - All details
```

### Inputs
```
Background: bg-slate-900
Border: border border-slate-600 focus:border-blue-500
Text: text-white placeholder-slate-400
Radius: rounded-lg
```

### Badges
```
Team: bg-blue-600 text-white px-2 py-1 rounded
Points: bg-green-600 text-white
Negative: bg-red-600 text-white
Warning: bg-yellow-600 text-black
```

---

## Animations

- **Page Transitions:** Fade + slide (200ms ease-out)
- **Card Hover:** Scale 1.02 + shadow (150ms)
- **Button Press:** Scale 0.98 (100ms)
- **Live Updates:** Pulse animation on new data
- **Draft Timer:** Countdown with urgency colors
- **Points Scored:** Counter animation + glow
